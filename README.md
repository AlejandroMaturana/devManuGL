## 🌟 Características de la Misión

Esta es mi vitrina; una plataforma de **Mission Control** que integra:

- 🎨 **Estética Neo-Terminal**: Interfaz moderna y elegante inspirada en consolas de comando, con una cuadrícula de terminal ("Terminal Grid") y animaciones sutiles tipo _flicker_.
- 📱 **Arquitectura Responsiva**: Navegación lateral fija para equipos de escritorio y una barra de navegación tipo _overlay_ para dispositivos móviles.
- 📂 **Log de Proyectos (Mission Logs)**: Repositorio táctico que destaca mis trabajos estrella como _GourmetGo_ y _KanbanPro_, indicando fechas de despliegue y tecnologías clave.
- 🛡️ **Canal de Comunicación Seguro**: Formulario de contacto integrado mediante **Formspree**, optimizado para recibir señales y datos (_payloads_) de potenciales colaboradores.
- 📊 **Métricas de Perfil**: Visualización de habilidades en Frontend (85%) y Backend (92%), además de mi trasfondo como Ingeniero Industrial.

---

## 🛠️ Stack Tecnológico

La arquitectura de este portafolio utiliza herramientas de vanguardia para asegurar eficiencia y una experiencia visual premium:

- **Frontend**: HTML5 Semántico + JavaScript ES6+ (Lógica dinámica y configuración de temas).
- **Estilos**: **Tailwind CSS** para un diseño atómico y moderno, con fuentes de Google (**Space Grotesk** e **Inter**).
- **Herramientas**: Git/GitHub, Formspree para mensajería y Google Material Symbols para iconografía técnica.
